## UI Components вЂ” `@repo/shadcn`

UI primitives (Button, Input, etc.) live in `packages/ui` as shadcn/ui components.
Import directly вЂ” no barrel files:

```tsx
import { Button } from "@repo/shadcn/components/Button";
import { cn } from "@repo/shadcn/lib/utils";
```

Do NOT create `components/ui/` inside `apps/web`. All UI primitives come from `@repo/shadcn`.
